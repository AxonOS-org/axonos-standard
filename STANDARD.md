# AxonOS Standard

This document is the high-level entry point for the AxonOS Standard.

The canonical draft material lives in `standard/AOS-0000` through
`standard/AOS-0011`. Those numbered artifacts define the active Draft 0.1
technical standard sequence.

The standard covers:

- system boundary between neural hardware and applications;
- canonical terminology;
- evidence levels and claims discipline;
- neural permissions;
- consent semantics;
- conformance profiles;
- typed intent events;
- IPC and timing contracts;
- security and privacy threat modelling;
- reference implementation mapping;
- governance and change control.

This file is an index. It is not a replacement for the numbered AOS artifacts.
