# AxonOS Standard

This file is the high-level index for the AxonOS Standard.

Normative draft material is maintained in `standard/AOS-0000` through `standard/AOS-0011`.
Those artifacts define the draft technical structure of AxonOS Draft 0.1.

The standard governs deterministic BCI system boundaries, neural permissions,
consent semantics, typed intent events, evidence levels, conformance profiles,
security and privacy threat modelling, reference implementation mapping, and
governance and change control.
