# Conformance

Status: Draft.

AxonOS conformance is profile-based. The initial profiles are defined in
`standard/AOS-0006-conformance-profiles.md`.

Draft profiles:

- S0 — documentation alignment;
- S1 — SDK boundary compatibility;
- K1 — kernel substrate compatibility;
- C1 — consent semantics compatibility;
- V1 — evidence-tagged validation.

These profiles are not certification classes and do not imply regulatory
approval. They are a vocabulary for implementation review.
