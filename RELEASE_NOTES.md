# AxonOS Standard Draft 0.1.1

Draft 0.1.1 is a repository-hardening release.

## Scope

This release focuses on:

- green CI;
- seven independent verification jobs;
- chronological AOS artifact integrity;
- Markdown readability;
- license clarity;
- claims-register hygiene;
- legacy directory redirects;
- release contract validation.

## Status

Pre-normative draft. This is not a final conformance standard, not a clinical
protocol, and not a regulatory approval claim.
