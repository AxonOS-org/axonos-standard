# Security Policy

AxonOS treats neural data security as a primary system property.

## Reporting

Security reports should be sent to:

security@axonos.org

Please include:

- affected repository;
- affected component;
- reproduction steps;
- impact assessment;
- suggested mitigation if available.

## Scope

Security-relevant issues include:

- neural data exposure;
- capability bypass;
- consent bypass;
- timing side channels;
- unsafe memory behavior;
- incorrect validation claims;
- supply-chain risks.

## Public disclosure

Coordinated disclosure is preferred. Do not disclose vulnerabilities publicly before maintainers have had a reasonable opportunity to respond.
